Experience the game directly in your browser! Our web build leverages WebAssembly (WASM) for high-performance physics calculations via the Rapier engine.

### Technical Highlights
- **Engine**: React Three Fiber (Three.js)
- **Physics**: Rapier 3D (Compiled to WebAssembly)
- **State Management**: Zustand
- **Graphics**: PBR Materials, Volumetric Clouds, Dynamic Biomes
- **Performance**: 60 FPS target on mobile and desktop
